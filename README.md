## Getting Started

---

#### Environment File

Create a file called `.env.local` at the root of the project and fill your Snipcart API:

```
NEXT_PUBLIC_SNIPCART_API_KEY="[Your Snipcart API Key]"
```

## Running locally

---

Start your local server:

```bash
npm install
npm run dev
# or
yarn dev
```
